# Frontend on React + TypeScript

## How to set up:
1. Clone repo:
```bash
git clone https://github.com/Miryz21/DevOps-2026.git
cd DevOps-2026/frontend
```
2. Make sure [npm](https://docs.npmjs.com/about-npm) installed: `npm -v`
3. Set up node_modules with npm: `npm install`
4. Run dev server: `npm run dev`
